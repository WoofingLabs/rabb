XFist DApp 白皮书

一、项目概述

XFist DApp 是一个基于 X Layer 网络（chainId: 196）的去中心化应用，旨在通过质押和推荐机制激励用户参与网络建设与生态发展。核心代币 XFIST 结合节点机制和奖励系统，为用户提供公平、透明的参与机会，推动去中心化网络的繁荣。

项目部署在 X Layer 网络，智能合约地址为 0x0bbCeFEfC27b967c6f447E72eD952B1Bf222fa1B，通过 MetaMask 钱包与 X Layer 网络交互。本白皮书详细介绍 XFist DApp 的规则与机制。

二、代币机制

XFist DApp 的核心代币为 XFIST，遵循 ERC-20 标准，总供应量上限为 1,000,000,000,000,000 XFIST。主要用途如下：

  • 质押：用户质押 XFIST 到节点，参与 XFIST 奖励分配。
  • 奖励：质押、节点和推荐行为获得 XFIST 奖励，推荐人额外获得 OKB 奖励（X Layer 主网币）。
  • 费用：加入网络和成为节点需支付 OKB。

三、加入网络

用户需支付 0.1 OKB 加入 XFist 网络，成为正式参与者。规则如下：

  • 推荐人机制：可提供有效推荐人地址（X Layer 有效地址，需已加入网络），若留空，系统将指定默认推荐人地址。
  • 费用：支付 0.1 OKB，进入奖励池并支持网络运营。
  • 状态：加入后可查看推荐信息、质押状态和待领取 XFIST 奖励（推荐奖励和质押奖励总和）。

加入网络是参与质押和节点功能的前提，未加入用户无法操作。

四、节点机制

用户可支付 1 OKB 升级为节点，获得更高奖励权限。规则如下：

  • 费用：支付 1 OKB，进入奖励池并支持网络维护。
  • 节点 ID：分配唯一 ID（最多 100,000 个节点），用于接受质押和奖励分配。
  • 权限：节点可接受其他用户 XFIST 质押，获得 10% 质押奖励分成（XFIST）。
  • 要求：需先加入网络（支付 0.1 OKB）。

成为节点适合深度参与 XFist 生态的用户，可显著提升收益。

五、质押规则

用户通过质押 XFIST 参与奖励分配，规则如下：

  • 最低质押量：每次质押为 10,000 XFIST 的整数倍（如 10,000、20,000 等）。
  • 节点选择：需指定有效节点 ID（由注册节点提供）。
  • 质押操作：通过 DApp 输入节点 ID 和质押数量，或选择余额的 25%、50%、75%、100% 快速质押。
  • 提取质押：随时提取 XFIST 本金及累积 XFIST 奖励，无锁定期。
  • 限制：用户只能质押到一个节点，且不能为合约地址。

质押是核心参与方式，质押越多，奖励越高，提取灵活无额外费用。

六、奖励机制

XFist DApp 通过 XFIST 奖励激励质押和节点参与，通过 OKB 奖励激励推荐行为。具体规则如下：

奖励类型 规则 具体数字
待领取奖励 实时显示累积 XFIST 奖励（推荐奖励 + 质押奖励），精确到 2 位小数 例如：123.45 XFIST
质押奖励（XFIST） 每 10,000 XFIST 每区块奖励，基于当前奖励基数 初始：0.00025 XFIST/区块

推荐奖励（XFIST） 推荐人每区块获基础奖励 + 10% 推荐加成/推荐人数 初始：5 XFIST/区块 + 0.5 XFIST/推荐人
例：1 推荐人，获 5.5 XFIST/区块
推荐奖励（OKB） 推荐人获新用户加入费用（0.1 OKB）的 3%-0.5%（10 层递减），最大 10,000 OKB/用户 第 1 层：0.03 OKB

奖励由智能合约透明计算，基于 xFistPerBlock 和 getCurrentStakeReward 动态调整，用户可通过 DApp 查看累积 XFIST 奖励。

七、减产机制

XFist DApp 采用减产机制控制 XFIST 奖励产出，规则如下：

项目 规则 具体数字
初始奖励 每区块的 XFIST 奖励基数 推荐人：5 XFIST/区块
质押（每 10,000 XFIST）：0.0025 XFIST/区块
最低奖励 奖励基数的最低值 推荐人：0.1 XFIST/区块
质押（每 10,000 XFIST）：0.00005 XFIST/区块
减产周期 每 6,000,000 区块（约 27.78 天，0.4 秒/区块）减产一次 6,000,000 区块
减产幅度 每周期奖励基数减少，公式：base = 50 - (effectiveBlocks / 6,000,000) 每周期减少约 2%（1/50）
减产计划 示例减产阶段（每 6,000,000 区块）：
  • 第 1 周期（0-6M 区块）：5 XFIST，0.0025 XFIST（每 10,000 XFIST）
  • 第 2 周期（6M-12M 区块）：4.9 XFIST，0.00245 XFIST
  • 第 3 周期（12M-18M 区块）：4.8 XFIST，0.0024 XFIST

 最大 49 次 减产
查询方式 通过 DApp 或区块浏览器查看减产进度 调用 getReductionProgress

减产机制控制 XFIST 通胀，激励早期参与，维持生态长期稳定。

八、推荐系统

推荐机制激励用户邀请新用户，规则如下：

  • 推荐奖励（OKB）：推荐人获新用户加入费用（0.1 OKB）的 3%-0.5%（10 层递减），分别为 0.03、0.02、0.01、0.01、0.005、0.005、0.005、0.005、0.005、0.005 OKB，单用户最大 10,000 OKB。
  • 推荐奖励（XFIST）：推荐人每区块获基础奖励（初始 5 XFIST）+ 10% 加成/推荐人数。
  • 推荐信息：显示推荐人数和推荐人地址（无推荐人显示“无”）。
  • 默认推荐人：用户可留空推荐人地址，系统将自动分配默认推荐人。
  • 层级限制：最多 10 层推荐路径，防止循环引用，确保公平分配。

推荐系统通过 XFIST 和 OKB 奖励激励社区增长。

九、注意事项

为顺利参与 XFist DApp，请注意：

  • 钱包要求：需 MetaMask 钱包，连接 X Layer 网络（chainId: 196，RPC: https://rpc.xlayer.tech）。
  • 费用准备：加入网络需 0.1 OKB，节点需 1 OKB，gas 费约 0.001-0.005 OKB/交易。
  • 网络环境：通过本地服务器运行 DApp（例如：npx serve）。
  • 状态检查：操作后状态可能因网络延迟稍迟，建议等待数秒刷新。
  • 合约透明：数据可通过 X Layer 区块浏览器（https://www.okx.com/web3/explorer/xlayer）验证。

十、总结

XFist DApp 通过质押和推荐机制激励用户参与 X Layer 生态建设。用户可通过加入网络（0.1 OKB）、成为节点（1 OKB）、质押 XFIST（最低 10,000 XFIST）及邀请新用户获得 XFIST 奖励，推荐人额外获得 OKB 奖励。灵活的提取机制和透明的减产计划确保公平收益，推动去中心化金融发展。
